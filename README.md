# Portfolio SABRI Zakaria

Ceci est le dépôt de mon portfolio personnel, développé dans le cadre de ma formation en **BUT Informatique**. Ce site a pour objectif de présenter mes réalisations, personnelles mais aussi professionnelles via les recueils de **SAÉ (Situation d'Apprentissage et d'Évaluation)** validés durant mon cursus.

## Aperçu
Le portfolio est une application web moderne, conçue avec une approche **"Clean Code"** sans frameworks lourds, privilégiant la performance et la simplicité.

### Points clés :
- **Architecture :** HTML5 Sémantique & CSS3 (Variables, Flexbox, Grid).
- **Interactivité :** JavaScript Vanilla (ES6+) pour l'injection dynamique des projets et la gestion d'une interface modale.
- **Design :** Thème sombre (Dark Mode) avec la police Montserrat.
